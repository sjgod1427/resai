## 🚀 Quick Command Reference

### First-Time Setup

```bash
cd bias_recommender

# Install dependencies
pip install -r requirements.txt

# Create .env file
cp .env.example .env
# Edit .env and add your GROQ_API_KEY

# Build data files
python dataset_builder.py      # Downloads + processes YouTube data
python user_profiles.py        # Generates 20 personas
```

### Running the Application

```bash
# Web UI (recommended)
python flask_app.py            # Open http://localhost:5000

# Or offline study
python main.py                 # Generates plots and comparison report

# Or legacy Gradio UI
python app.py                  # Old interface (Gradio-based)
```

### Testing

```bash
# Architecture tests (no LLM)
python test_architecture.py    # ~15 sec

# LLM correction tests
python test_llm_correction.py  # ~30 sec, requires GROQ_API_KEY
```
